29 Problem 1: Confidence intervals from a butterfly field study

29.3 Step 2: Summary statistics

Compute the sample mean and sample standard deviation.

n <- length(wingspan)
xbar <- mean(wingspan)
s <- sd(wingspan)

For a Z-confidence interval, we need

sigma <- 1.6
se_z <- sigma / sqrt(n) # SE is SD/sqrt(n)

For a t-confidence interval, we need

df <- n - 1
se_t <- s / sqrt(n)

29.4 Step 3: Identify confidence level and look up critical-value

For a two-sided 95% interval, \(\alpha = 0.05\):

lookup_table <- data.frame(
  method = c("Z", "t"),
  probability_to_left = c(0.975, 0.975),
  degrees_of_freedom = c(NA, 15),
  critical_value = c(qnorm(0.975), qt(0.975, 15))
)

lookup_table
  method probability_to_left degrees_of_freedom critical_value
1      Z               0.975                 NA       1.959964
2      t               0.975                 15       2.131450

A traditional table lookup gives approximately:

  • \(z_{0.975} = 1.96\);
  • \(t_{0.975,15} = 2.131\).

29.5 Step 4: Plug in numbers

For a 95% Z-confidence interval, we use \(z_{0.975}\).

ci_z <- c(xbar - 1.96 * se_z,
          xbar + 1.96 * se_z)

ci_z
[1] 50.45975 52.02775

For a 95% t-confidence interval, we use \(t_{0.975, 15}\).

ci_t <- c(xbar - 2.131 * se_t,
          xbar + 2.131 * se_t)

ci_t
[1] 50.38781 52.09969

30.4 Step 3: Summary statistics

First compute the sample means for the two habitats.

xbar1 <- mean(habitat_A)
xbar2 <- mean(habitat_B)

The observed mean difference is

diff_means <- xbar1 - xbar2
diff_means
[1] 3.060417

Note that degree of freedom and standard error are more complicated, we have introduced two methods to estimate them in class – Welch’s unpooled standard error, and pooled standard error. Welch’s method is usually the safer default because it does not require equal population variances. The pooled test can be reasonable when the two groups are believed to have similar variances, either from scientific knowledge or from the data.

In this exercise, we focus on using the Welch’s unpooled standard error,

\[SE_{\text{Welch}} = \sqrt{\frac{s_1^2}{n_1} + \frac{s_2^2}{n_2}}.\]

The Welch degrees of freedom are approximated by

\[df = \frac{\left(\frac{s_1^2}{n_1} + \frac{s_2^2}{n_2}\right)^2} {\frac{(s_1^2/n_1)^2}{n_1-1} + \frac{(s_2^2/n_2)^2}{n_2-1}}.\] To compute the standard error, we also need sample sizes and standard deviations from the data:

n1 <- length(habitat_A)
n2 <- length(habitat_B)

s1 <- sd(habitat_A)
s2 <- sd(habitat_B)

data.frame(
  habitat = c("habitat_A", "habitat_B"),
  n = c(n1, n2),
  mean = c(xbar1, xbar2),
  sd = c(s1, s2)
)
    habitat  n     mean       sd
1 habitat_A 16 51.24375 1.606640
2 habitat_B 12 48.18333 1.410888

Then we can put everything together and compute the standard error:

se_welch <- sqrt(s1^2 / n1 + s2^2 / n2)

df_welch <- (s1^2 / n1 + s2^2 / n2)^2 /
  ((s1^2 / n1)^2 / (n1 - 1) + (s2^2 / n2)^2 / (n2 - 1))

tcrit_welch <- qt(0.975, df = df_welch)

30.5 Step 4: Plug in numbers

For our 95% confidence interval, we have

ci_welch <- c(diff_means - tcrit_welch * se_welch,
              diff_means + tcrit_welch * se_welch)
ci_welch # confidence interval
[1] 1.882947 4.237886

For our t-test statistics, we have

t_welch <- diff_means / se_welch
p_welch <- 2 * (1 - pt(abs(t_welch), df = df_welch))

t_welch # test statistics
[1] 5.350129
p_welch # p-val
[1] 1.460313e-05

t.test(habitat_A, habitat_B, alternative = "two.sided", var.equal = FALSE)

30.7 Conclusion

Both approaches answer the same question: is the mean wingspan different between the two habitats?

The confidence-interval approach and the p-value approach agree:

  • If the 95% confidence interval excludes 0, then the two-sided p-value is below 0.05.
  • If the 95% confidence interval includes 0, then the two-sided p-value is above 0.05.
